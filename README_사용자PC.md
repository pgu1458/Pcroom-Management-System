# 사용자 PC (User Client)

피씨방 통합 관리 시스템의 **사용자 PC 클라이언트**입니다.
각 자리에 설치되어 로그인부터 게임 실행, 음식 주문, 관리자 채팅까지 지원합니다.

---

## 주요 기능

### 로그인 / 회원가입 (UserPcLogin)
- 아이디 / 비밀번호 입력 후 9000 포트로 인증 요청
- 엔터 키 로그인 지원
- 로그인 성공 시 이름, 남은 시간, 좌석 번호 팝업 후 메인 화면으로 전환
- 회원가입 팝업: 이름, 생년월일, 아이디, 비밀번호, 전화번호 입력
- 전체 화면 고정, ESC로 종료

### 메인 화면 (UserPcMain)
**이용 정보**
- 이름, 좌석 번호 표시
- 남은 이용 시간 실시간 카운트다운 (HH:MM:SS)
- 30초마다 서버와 시간 자동 동기화 (키오스크 충전 즉시 반영)
- 시간 만료 시 알림 메시지

**게임 런처** (바로가기 버튼)
- 로스트아크, 메이플스토리, 배틀그라운드, 피파온라인, 서든어택
- 리그오브레전드, 오버워치2, 스팀 런처, OP.GG

**PC 설정 바로가기**
- 마우스 설정, 키보드 설정, 마우스 속도
- 야간 모드, 게임 모드, 사운드 설정, 제어판

**음식 주문**
- 주문 버튼 클릭 → 메뉴 폼(Menu) 실행

**관리자 문의**
- 문의 버튼 클릭 → 채팅 폼(ChatForm) 실행

**종료 / 로그아웃**
- 현재 남은시간을 서버에 저장 후 로그아웃
- 로그인 화면으로 복귀

### 음식 주문 (Menu)
- 카테고리별 탭 구성

| 카테고리 | 메뉴 수 |
|----------|---------|
| 면류 | 17종 |
| 밥류 | 8종 |
| 분식 | 8종 |
| 스낵 | 10종 |
| 음료 | 16종 |

- 메뉴 버튼 클릭 시 장바구니(ListView)에 추가
- 같은 메뉴 클릭 시 수량 자동 증가
- 더블 클릭으로 장바구니 항목 삭제
- 총액 실시간 계산
- 주문 전송 시 9001 포트로 체크섬 포함 전송
- 주문번호, 총액 팝업 후 장바구니 초기화

### 관리자 채팅 (ChatForm)
- 관리자에게 메시지 전송 (9000 포트)
- 1.5초마다 관리자 답장 자동 수신 (폴링)
- 시간 표시 포함 채팅 UI

## 프로젝트 구조

```
UserPC/
├── Program.cs              # 진입점
├── Form2.cs                # 메인 폼 (전체화면, ESC 종료)
├── UserPcLogin.cs          # 로그인 / 회원가입
├── UserPcMain.cs           # 메인 화면 (시간, 게임, 설정, 주문, 채팅)
├── Menu.cs                 # 음식 주문 폼
├── ChatForm.cs             # 관리자 채팅
├── TcpClientService.cs     # TCP 클라이언트 싱글톤 (9000/9001 포트)
├── Checksum.cs             # 체크섬 유틸
└── Images/                 # 메뉴 이미지 폴더
```
