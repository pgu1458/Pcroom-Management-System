# 관리자 PC (Admin Server)

피씨방 통합 관리 시스템의 **서버 겸 관리자 클라이언트**입니다.
프로그램 실행과 동시에 TCP 서버(9000/9001/9002 포트)가 자동으로 시작됩니다.

---

## 주요 기능

### 로그인 (CounterLogin)
- 아이디 / 비밀번호로 로그인
- Member.db의 role 컬럼으로 권한 구분
  - **관리자** → 관리자 화면(Admin)으로 이동
  - **알바** → 알바 화면(CounterAl)으로 이동
  - **손님** → "키오스크를 이용해주세요" 안내

### 알바 화면 (CounterAl)
실시간으로 서버 이벤트를 구독해서 UI를 자동 갱신합니다.

**좌석 관리**
- 50개 좌석 실시간 상태 표시 (이름, 이용시간, 남은시간)
- 좌석 클릭 시 해당 회원 상세 정보 ListView로 표시
- 1초 타이머로 이용시간/남은시간 카운트
- 현재 이용 중인 좌석 수 실시간 표시
- 키오스크 충전 완료 즉시 남은시간 라벨 갱신

**주문 관리**
- 사용자 PC에서 주문 수신 시 자동으로 그리드에 추가
- 주문번호, 메뉴 내역, 총액, 시간 표시
- Food.db 재고/판매량 자동 갱신

**회원 관리**
- 전체 회원 목록 조회 (dataGridView3)
- 전화번호 뒷자리로 회원 검색
- 회원 정보 직접 수정 및 저장
- 회원 삭제

**음식 재고 관리**
- Food.db 메뉴별 재고/판매량 조회 및 수정

**채팅**
- 사용자 PC에서 보낸 문의 수신
- 좌석 선택 후 해당 이용자에게 답장 전송

**로그**
- TCP 통신 로그 실시간 출력

### 관리자 화면 (Admin)
- 월/일 선택으로 매출 차트 조회 (일별/시간대별 막대 차트)
- 총 매출 금액 표시
- Food.db 메뉴 데이터 직접 수정 및 저장

---

## 서버 구조 (TcpServer)

싱글톤 패턴으로 구현된 TCP 서버입니다. 3개의 포트를 동시에 리슨합니다.

```
TcpServer.Instance.Start()   ← Form1 생성자에서 자동 호출
TcpServer.Instance.Stop()    ← 프로그램 종료 시 자동 호출
```

각 포트별 처리 내용은 아래와 같습니다.

**9000 포트 (사용자 PC)**
- 로그인 시 빈 좌석 자동 배정
- 회원가입 처리 및 DB 저장
- 로그아웃 시 남은시간 DB 저장 + 좌석 반환
- 30초마다 사용자 PC 남은시간 동기화 응답
- 채팅 메시지 수신/전달

**9001 포트 (음식 주문)**
- 주문 수신 후 Food.db 재고/판매량 갱신
- sales.db에 매출 데이터 저장
- ACK 응답 전송

**9002 포트 (키오스크)**
- 연결 시 즉시 전체 좌석 현황 푸시
- 충전 요청 처리 및 DB 갱신
- 이용 중인 회원 충전 시 관리자 화면 실시간 반영

## 프로젝트 구조

```
Admin/
├── Program.cs          # 진입점 (Form1 실행)
├── Form1.cs            # 메인 폼 (TcpServer 시작, 화면 전환)
├── CounterLogin.cs     # 로그인 화면 (관리자/알바 구분)
├── CounterAl.cs        # 알바 화면 (좌석/주문/채팅/회원관리)
├── Admin.cs            # 관리자 화면 (매출 차트, Food 관리)
├── TcpServer.cs        # TCP 서버 싱글톤 (9000/9001/9002)
├── Checksum.cs         # 체크섬 유틸
├── Member.db           # 회원 DB
├── Food.db             # 음식 메뉴/재고 DB
└── sales.db            # 매출 DB
```

## 실행 방법

1. Visual Studio에서 프로젝트 열기
2. NuGet 패키지 설치
   ```
   System.Data.SQLite
   System.Windows.Forms.DataVisualization (차트용)
   ```
3. 빌드 후 실행
4. 기본 관리자 계정으로 로그인 (Member.db에서 role = '관리자' 계정 필요)
